🏗️ 架构设计
────────────────────────────────────────────────────────────────────────────────

后端分层架构
  Controller (REST API) ↓
     ↓
  Service (业务逻辑 + 缓存) ↓
     ↓
  Mapper (数据访问) ↓
     ↓
  MySQL (数据库)
     ↑
  WebSocket Handler ← OnlineStatusManager

前端架构
  Components (ChatWindow.vue, ChatList.vue)
     ↓
  Vuex Store (chat.js)
     ↓
  HTTP API (chat.js) + WebSocket Client (websocket.js)
     ↓
  Backend Services

设计亮点
  ✅ 分层清晰 - 职责单一
  ✅ 可选集成 - @Autowired(required=false)
  ✅ 缓存优化 - @Cacheable/@CacheEvict
  ✅ 错误处理 - 统一Result返回
  ✅ 并发安全 - ConcurrentHashMap
  ✅ 双通道 - WebSocket优先+HTTP备选
